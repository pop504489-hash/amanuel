# **App Name**: Wholesale Connect

## Core Features:

- Visual Product Catalog: A grid-based product display with large images, clear product names, and pricing for easy browsing by non-tech-savvy users.
- One-Tap Quick Ordering: Simple '+' and '-' buttons directly on the product catalog for quickly adjusting item quantities and adding/removing them from the cart.
- Order Review & Checkout: A streamlined cart and checkout process, allowing users to review selected items, adjust quantities, and confirm their order efficiently.
- Order Tracking Dashboard: A user-friendly dashboard showing the current status of past orders (e.g., 'Pending', 'Out for Delivery', 'Delivered').
- Smart Natural Language Search: An AI-powered tool that understands natural language queries (e.g., 'I need 2 crates of soft drinks') to quickly find products.
- Basic Inventory Forecasting: An AI tool that provides simple reorder suggestions based on the shop's past purchasing patterns to optimize inventory levels.
- Multilingual User Interface: Allows users to switch between English and Amharic for all application interface text, enhancing accessibility.

## Style Guidelines:

- Primary color: A deep, professional blue (#123EB4) to convey trust and reliability, consistent with B2B operations.
- Background color: A very light, desaturated blue (#EAEFF6) providing a clean, spacious canvas that complements the primary blue.
- Accent color: A vibrant yet clean sky blue (#4DC9F3) to highlight interactive elements, calls to action, and important notifications, ensuring high visibility.
- Body and headline font: 'Inter', a neutral, modern sans-serif typeface, chosen for its excellent readability and clean aesthetic across various text sizes.
- Use clear, intuitive icons with strong visual metaphors to represent actions and data (e.g., shopping cart, truck for delivery status, magnifying glass for search). Icons should be easily understandable by non-tech-savvy users.
- Implement a 'Visual-First' design strategy with generous white space. Products will be displayed in large, consistent grid tiles, ensuring all crucial information is immediately visible without clutter.
- Subtle, fluid animations for interactive elements (e.g., adding items to cart, navigation transitions) to provide immediate feedback and enhance the perceived responsiveness and smoothness of the user experience.
